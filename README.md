KAde Suite
KAde Suite è un’estensione Chrome per scaricare, organizzare e archiviare localmente F24 e Certificazioni Uniche dal Cassetto Fiscale dell’Agenzia delle Entrate.
È pensata per CAF, studi commerciali, consulenti del lavoro, professionisti fiscali, intermediari abilitati e uffici amministrativi che gestiscono frequentemente documenti fiscali per più soggetti.
Funzioni principali
download massivo degli F24 disponibili in formato PDF;
download massivo delle Certificazioni Uniche;
unione automatica degli F24 composti da più PDF o quietanze;
rinomina automatica dei documenti;
riconoscimento dei file già scaricati;
salvataggio ordinato nelle cartelle `Download/KAdeSuite/F24` e `Download/KAdeSuite/CU`;
elaborazione locale, senza invio dei documenti a server esterni dello sviluppatore.
Requisiti d’uso
Per utilizzare KAde Suite è necessario essere già autenticati sul sito dell’Agenzia delle Entrate e aver selezionato l’utenza di lavoro o il soggetto corretto.
Per gli F24 occorre trovarsi nella pagina F24 / Versamenti. Per le Certificazioni Uniche occorre trovarsi nella relativa pagina del Cassetto Fiscale.
KAde Suite può scaricare gli F24 solo quando il sito li rende effettivamente disponibili in formato PDF. Questa condizione è tipica delle utenze Entratel o degli intermediari abilitati e non è sempre presente per l’utente finale.
